================================================================================
COMPILATION INSTRUCTIONS
================================================================================

Standard Compilation:
$ xelatex CV_filename.tex

Clean Compilation:
$ xelatex CV_filename.tex && rm -f *.aux *.log *.out *.fls *.fdb_latexmk *.gz

IMPORTANT: 
- Always use xelatex (NOT pdflatex)
- Always use \moderncvstyle{fancy} (NOT casual)
- Clean up artifacts after compilation

================================================================================
COMMON MISTAKES TO AVOID
================================================================================

1. Using \moderncvstyle{casual} (causes multi-page errors)
2. Inconsistent date formatting
3. Missing metrics in achievements
4. Passive voice in bullet points
5. Wrong number of \cventry arguments (must be 6)
6. Forgetting \recomputelengths after geometry changes
7. Using pdflatex instead of xelatex
8. Incorrect color definitions
